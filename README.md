# @zeitport/eslint-parser-json

A JSON parser for [ESLint].

## Install

```
npm install @zeitport/eslint-parser-json
```

## Usage

[ESLint] configuration
```
{
    "parser": "@zeitport/eslint-parser-json,
    "rules": {
    }
}
```
