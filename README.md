# eslint-json-parser
A JSON parser for [ESLint].

## Install

```
npm install @zeitport/eslint-json-parser
```

## Usage

[ESLint] configuration
```
{
    "parser": "@zeitport/eslint-json-parser,
    "rules": {
    }
}
```
